Imperial Hotels

Une application React JS / Flux pour cette chaîne d'hotel londonienne.

On m'a demandé de créer cette interface pour une application admin pour les promotions et packages de cette chaîne d'hotels londonienne.

Se connectant à une API REST, j'ai choisi d'utiliser la combinaison de React JS et de l'architecture Flux pour bâtir cette application simple mais performante.

L'application est constituée de quelques écrans simples pour lister les éléments, en créer de nouveaux et éditer ou supprimer des éléments existants.

J'ai également mis en place les divers environnements de dévelopement sur la plateforme heroku et ai utilisé certains modules du framework Bootstrap afin de créer les composants des divers formulaires.
